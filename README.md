# Desenvolvimento-de-Sistemas

## Sobre o Projeto

Este projeto consiste no desenvolvimento de uma **API REST utilizando C# e .NET**, com o objetivo de demonstrar na prática o funcionamento da **arquitetura cliente-servidor** e o uso do **protocolo HTTP** para comunicação entre aplicações.

A API foi construída utilizando **ASP.NET Core Web API** e disponibiliza endpoints que retornam **dados fictícios de produtos** em formato **JSON**. Esses dados podem ser acessados por meio de requisições HTTP, permitindo que diferentes aplicações clientes consumam as informações fornecidas pelo servidor.

Para facilitar o teste e a visualização dos endpoints disponíveis, foi utilizada a ferramenta **Swagger / OpenAPI**, que permite interagir com a API diretamente pelo navegador, realizando requisições e analisando as respostas retornadas pelo servidor.

Durante o desenvolvimento do projeto, foram aplicados conceitos fundamentais do desenvolvimento de aplicações modernas, como **criação de APIs REST, manipulação de requisições HTTP, organização de endpoints e estruturação de projetos em .NET**. Dessa forma, o projeto demonstra como serviços web podem ser utilizados para permitir a comunicação e a integração entre diferentes sistemas.
